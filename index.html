Introduction

ZweLess is a LESS library that includes a lot of mixins and variables to help you styling a website. Nowadays there are a lot of different browsers, each with its set of prefixed properties and incompatibilities. ZweLess tries (and we can say succeded quite well) to think about that for you, saving you time to enjoy the styling process and enhance your creativity.

Installation

Start using ZweLess is really easy: just download the project, copy the whole less directory into your website folder and include the zwe.less file: @import "zwe";.
Now you can start using all the mixins and variables provided by ZweLess

Variables

CSS doesn't have variables, but LESS does, and we try to take advantages of this feature to provide everything a designer wants.

Colors

Variables for colors could be too much, so we provide only grayscale: from #000 to #fff in steps of 10%: 
//Grayscale
@black-10: darken(#fff, 10%); // #e6e6e6
@black-20: darken(#fff, 20%); // #ccc
@black-30: darken(#fff, 30%); // #b3b3b3
@black-40: darken(#fff, 40%); // #999
@black-50: darken(#fff, 50%); // #808080
@black-60: darken(#fff, 60%); // #666
@black-70: darken(#fff, 70%); // #4d4d4d
@black-80: darken(#fff, 80%); // #333
@black-90: darken(#fff, 90%); // #191919
For example one of the above squares is colorized in this way: 
.black-10 {
    background: @black-10;
}
.black-10 {
    background: #e6e6e6;
}
