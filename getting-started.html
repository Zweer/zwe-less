What is ZweLess?

ZweLess is a collection of Less mixins and classes that provide a big help to designers and coder which want to stylize their sites quickly.

To use ZweLess at its full power you have to understand and talk Less to use all the mixins the library provide.
You'll quickly see that everything is organized in a OO way, so that everything is exactly where you expect it to be.

Moreover ZweLess wants (or better tries) to be an example of good OO css programming, taking in action all the experience we have built in years of web development.

What ZweLess is not (and never will be)?

ZweLess doesn't want to be a complete framework with also javascript and templates in it. We leave that to the incredible Bootstrap.

Also ZweLess doesn't want to be the Bible of css programming (if we can call it "programming") or styling. Again, we leave it to bigger projects with more people and a wider audience.

Download ZweLess

ZweLess could be used in many ways, each with its very flexibility. Be aware that not everyone may have the knowledge or resources to use a way or another.

Compiled CSS

The easiest way to use this library is to download the compiled CSS. In this way you can benefit all the resets and settings made for you, without using Less or other pre-processors.

Obviously in this way you'll not take benefits from the power of Less and the universe of mixins the library provides.
